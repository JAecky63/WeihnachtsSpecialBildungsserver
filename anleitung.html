HTML-Weihnachtskarte – Anleitung

1. Grundstruktur einer HTML-Seite

Jede HTML-Seite beginnt mit einem festen Gerüst:

<!DOCTYPE html>
<html lang="de">
  <head>
    <meta charset="UTF-8">
    <title>Meine Weihnachtskarte</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <!-- Inhalt -->
  </body>
</html>

2. Verwendung von Stylesheets

Das Stylesheet style.css sorgt für Farben, Schriftarten und Layout. Es wird im <head> eingebunden:

<link rel="stylesheet" href="style.css">

Beispielhafte CSS-Regeln:

body {
  background-color: #286b92;
  color: white;
  font-size: 30px;
  text-align: center;
}

p.gross {
  font-size: 36px;
  font-weight: bold;
}

3. Einbindung von Icons mit FontAwesome

FontAwesome bietet viele schöne Symbole – z. B. 🎁 🎄 🛷

Einbindung im <head>:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

Verwendung im <body>:

<i class="fa-solid fa-gift" style="font-size: 100px; color: red;"></i>

4. Tabellen in HTML – einfach erklärt

Eine Tabelle ist wie ein Stundenplan: Sie besteht aus Zeilen (horizontal) und Spalten (vertikal). In HTML kannst du Tabellen verwenden, um Inhalte ordentlich nebeneinander darzustellen – zum Beispiel zwei Weihnachts-Icons.

🔧 Die wichtigsten Bausteine

  • <table> – Start der Tabelle
  • <tr> – Eine neue Zeile
  • <td> – Eine Zelle mit Inhalt
  • <th> – Eine Zelle mit Überschrift (fett)

🎄 Beispiel: Zwei Icons nebeneinander

<table>
  <tr>
    <td>
      <i class="fa-solid fa-sleigh" style="font-size: 100px; color: white;"></i>
    </td>
    <td>
      <i class="fa-solid fa-gift" style="font-size: 100px; color: red;"></i>
    </td>
  </tr>
</table>

🔍 Das bedeutet: Eine Zeile mit zwei Zellen – in jeder steckt ein Icon.

📅 Beispiel: Weihnachtszeitplan

<table>
  <tr>
    <th>Datum</th>
    <th>Aktivität</th>
  </tr>
  <tr>
    <td>24.12.</td>
    <td>Geschenke einpacken</td>
  </tr>
  <tr>
    <td>25.12.</td>
    <td>Familienessen</td>
  </tr>
</table>

🔍 Hier gibt es eine Überschriftenzeile und zwei Zeilen mit Daten.

🧠 Merktipp

  • T wie Table = Tabelle
  • TR wie Table Row = Zeile
  • TD wie Table Data = Zelle
  • TH wie Table Header = Überschrift

5. Links einfügen

Links verbinden deine Seiten oder führen zu externen Inhalten:

<a href="WeihnachtsGruss.html" target="_blank">HIER</a>

Du kannst auch Icons als Link verwenden:

<a href="WeihnachtsGruss_one.html">
  <i class="fa-solid fa-sleigh"></i>
</a>

6. Quellcode ansehen

Mit Rechtsklick → „Seitenquelltext anzeigen“ oder STRG + U kannst du den HTML-Code jeder Seite sehen und daraus lernen.
